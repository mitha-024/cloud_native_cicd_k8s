# Cloud-Native CI/CD Pipeline with Kubernetes

## Overview
Designed and implemented an end-to-end CI/CD pipeline using Jenkins, Docker, and Kubernetes (local cluster).

## Features
- Automated CI/CD pipeline using Jenkins
- Docker-based containerization
- Kubernetes deployment for scalability
- Infrastructure provisioning using Terraform
- GitHub webhook integration for automated triggers

## Tech Stack
- AWS (EC2)
- Docker
- Kubernetes (Minikube)
- Jenkins
- Terraform

## Workflow
1. Code is pushed to GitHub
2. Jenkins pipeline is triggered via webhook
3. Docker image is built
4. Application is deployed to Kubernetes cluster
